# منصة تهنئة عيد الأضحى

## رفع المشروع على GitHub Pages

1. ارفع جميع الملفات الموجودة في هذا المجلد إلى مستودع GitHub.
2. من إعدادات المستودع افتح `Settings`.
3. اختر `Pages`.
4. في `Build and deployment` اختر:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. احفظ الإعدادات.
6. افتح رابط GitHub Pages الذي يظهر لك.

## الملفات المهمة

- `index.html`
- `eid-adha.html`
- `eid-adha.css`
- `eid-adha.js`
- `templates.json`
- `assets/salati-eid-template.png`

ملفات الخادم المحلي اختيارية للتجربة على جهازك فقط:

- `local-server.ps1`
- `تشغيل-الخادم.bat`
- `local-server.js`
